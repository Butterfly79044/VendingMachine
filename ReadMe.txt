Реализован интерфейс администратора, подключение происходит по адресной строке https://localhost:7078/Admin/Panel
База данных создана через локальный SQL Server внутри visual studio через миграцию
Проект был выполнен в среде Visual Studio 2022

Класс Drink - отвечает за модель данных напитков
Класс DBContent - отвечает за подключение базы данных и ввод и тестовых данных для таблиц
Класс DrinkRepository - отвечает за функции с данными базы данных
